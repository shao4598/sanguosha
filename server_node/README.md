##  启动 Node.js 版服务端代码

0. 自行配置 Node.js 与 npm 环境。

1. 打开 server_node 文件夹。

2. 使用命令行下载第三方依赖库。

```shell
$ npm i
```

3. 拷贝客户端生成的 ```index.html```、```app.bundle.js```、```vendor.bundle.js``` 到本工程 ```views``` 目录下

4. 使用命令行启动主工程文件 index.js。

```shell
$ node index.js
```

5. 使用浏览器访问 ```http://localhost:3000```。
